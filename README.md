
# Slim

**A minimal skeleton for bootstrapping PHP projects**

*This project is a work in progress.*

___

### Features:

#### 🔋 Included batteries:
```
- Slim Framework
- Doctrine ORM
- PHPUnit
- Docker support (PHP-FPM + Nginx)
- Dependency injection with PHP-DI
- Environment variables support
```

#### 👷 Currently working on:
```
- Service container
- Database for Docker
```

#### ⏳ Roadmap:
```
- Middlewares
- JWT authentication
```
